# Genshin Tools 0.9.0 发布候选说明

此版本是 Go + Win32 重构后的首个内部发布候选，不是已授权公开发布版本。

## 主要能力

- 原生简洁暗色 Win32 外壳、托盘、单实例、DPI 和中英文界面。
- 键盘连按、鼠标左键连点和右键连点，包含停止键、焦点和会话安全策略。
- 原神安装发现、只读状态、纯净启动与启动参数。
- 游戏资源检查/下载/修复事务及断点恢复。
- 区服切换、HDR、声音和 BetterGI 固定协议联动。
- 截图、性能覆盖层、隔离注入 helper 和声明式插件生命周期。
- Ed25519 签名自更新、失败回滚和 FufuLauncher 上游只读差异报告。

## 永久排除

不包含项目自身登录、账号切换、米游社/HoYoLAB/BBS、抽卡统计、签到、资讯、数据中心、帮助文档、养成计算器、附加程序或内置浏览器。

## 已知发布限制

- 项目原创代码与文档已采用根目录 MIT License；发布包必须同时保留第三方许可证与通知。
- scope-v2 上游差异已逐项处置，基线已提升到 `5f6af35fcb90807d5db390ed4af58ca09ddd381c`。
- 真实游戏、反作弊、休眠/锁屏、RDP、混合权限、多显示器/DPI 和干净 Windows 10/11 矩阵尚需人工执行。
- 注入和插件默认关闭；只有通过来源、许可证、哈希、PE 和游戏版本审计的模块才允许启用。

自动验证及候选包 SHA-256 见 [S13 发布回归验证记录](s13-verification.md)。
