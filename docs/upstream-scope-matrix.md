# 上游功能范围矩阵

基线：FufuLauncher `master@b5a050ebd319341bddc4189491c90c22162d33fa`。  
规则版本：`scope-v2`（程序自更新/版本/安装器优先归入 `self-update`）。

本表是实施合同。自动对照上游时，已有模块按本表处理；无法匹配的新模块默认进入“待审查”，不会自动进入开发范围。

## 1. 导航与页面

| 上游入口/能力 | 决策 | 本项目处理 |
|---|---:|---|
| 首页启动按钮、游戏状态 | 保留 | 简化为状态摘要与启动操作 |
| 首页背景、视频背景、轮播、资讯卡 | 排除 | 不请求、不缓存、不显示 |
| 首页签到、便笺、账号 Widget | 排除 | 不注册相关服务 |
| 首页抽卡/成就/背包/玩家/视频/BBS Widget | 排除 | 不保留入口 |
| 注入设置 | 保留 | 独立高风险页面 |
| 控制面板/工具箱 | 排除 | 按用户要求整页及全部子功能删除 |
| 游戏设置 | 保留 | 删除账号切换区域和公告/兑换码内容 |
| 账号设置 | 排除 | 整页删除 |
| 其他功能 | 部分保留 | 只保留连点/连按；删除附加程序、浏览器 |
| 插件管理 | 保留 | 做来源、哈希、兼容和故障隔离 |
| 插件商店 | 保留并直接复用 Fufu 来源 | 固定连接 FufuLauncher 官方商店；不自建后端、不开放任意目录 URL，只处理游戏增强插件 |
| 数据中心 | 排除 | 整页及数据库删除 |
| 帮助文档 | 排除 | 整页删除 |
| VanillaBBS/BBS | 排除 | 整页及 WebView 删除 |
| 养成计算器 | 排除 | 整页删除 |
| 程序设置 | 部分保留 | 删除签到、账号、资讯、背景 API/WebView 设置 |

## 2. 游戏本地管理

| 功能 | 决策 | 验收边界 |
|---|---:|---|
| 自动发现游戏路径 | 保留 | 多候选时要求用户选择，不静默猜测 |
| 手动选择路径 | 保留 | 支持 Unicode、空格和长路径 |
| 游戏版本、区服、目录大小 | 保留 | 后台扫描并可取消；不冻结 UI |
| 自定义游戏 EXE 名称 | 保留 | 校验必须位于选定游戏目录 |
| 游戏运行状态 | 保留 | 处理快速退出和 PID 重用 |
| 普通启动 | 保留 | 失败返回 Win32 错误和可读说明 |
| 自定义命令行参数 | 保留 | 参数原样、转义规则有测试 |
| 分辨率预设/自定义宽高 | 保留 | 非法值不能写入或启动 |
| 窗口模式/无边框模式 | 保留 | 与上游行为对照后实现 |
| 目标显示器 | 保留 | Per-monitor DPI，显示器消失时回退默认 |
| 启动后最小化/托盘/退出 | 保留 | 单一状态机，不能竞态关闭 |
| HDR 开关和亮度 | 保留 | 修改前读取备份，失败不破坏注册表值 |
| 桌面快捷方式 | 保留 | 路径、工作目录、图标和参数正确 |
| 游戏账号列表/切换 | 排除 | 不读取或写入账号相关注册表/文件 |
| 公告、活动、兑换码 | 排除 | 不内嵌浏览或拉取内容 |
| 地图 | 排除 | 不保留页面 |

## 3. 资源管理

| 功能 | 决策 | 验收边界 |
|---|---:|---|
| 基础游戏下载 | 保留 | manifest 驱动、并发限制、进度、取消 |
| 语音包选择/下载 | 保留 | 语言映射和磁盘空间预检 |
| 预下载 | 保留 | 与正式包隔离，支持清理 |
| 校验与修复 | 保留 | 先报告计划，再写文件；哈希匹配才替换 |
| staging 后移动 | 保留并强化 | 同卷原子替换；跨卷使用复制+校验+替换 |
| 断点续传 | 计划增强 | 上游变化大，Phase 4 结合实际 API 确认 |
| 官服/B服快速切换 | 保留 | 备份 `config.ini` 和 SDK 文件，失败回滚 |
| 高级服务器转换 | 保留 | 单独确认变更清单，禁止不可逆批量删除 |

## 4. 输入、截图与覆盖层

| 功能 | 决策 | 验收边界 |
|---|---:|---|
| 键盘按住触发键连按目标键 | P0 保留 | 扫描码、down/up 成对、松开立即停 |
| 鼠标左键按住连点 | P0 保留 | 过滤注入事件，无自激回路 |
| 鼠标右键按住连点 | P0 保留 | 与左键独立验证 |
| 停止热键 | P0 保留 | 全局立即停并解除武装 |
| 连点间隔 | 保留并增强 | 默认 50ms，可配置且有安全下限 |
| 游戏窗口截图 | 保留 | 截图 hook 与输入 hook 共存 |
| 截图组合热键和目录 | 保留 | 处理冲突、不可写目录和重名 |
| FPS/CPU/GPU 覆盖层 | 保留 | 游戏退出后彻底释放线程、计时器和窗口 |

## 5. 注入与插件

| 功能 | 决策 | 验收边界 |
|---|---:|---|
| 启用/禁用注入 | 保留 | 关闭时走纯净普通启动路径 |
| 内置/独立注入模块选择 | 保留 | 适配接口版本化 |
| Launcher.dll / UnlockerIsland 适配 | 复用插件目录约定，不采用上游二进制 | 主仓库二进制无签名/版本且登记哈希不匹配；后续开源仓库仅作为 `Plugins/config.ini` 行为参考，项目继续使用隔离 helper |
| 插件发现、启停、重命名、排序 | 保留 | 文件操作限制在插件根目录 |
| 插件配置和预设 | 保留 | Schema 校验、原子保存、损坏回退 |
| 插件下载/修复/更新 | 保留 | HTTPS、哈希、签名/来源和回滚 |
| 插件商店搜索/排序/分页 | 保留 | 适配 Fufu API；商店不可推送范围外模块到主导航，下载验证走系统浏览器 |
| 头像插件、FPS 插件、主插件 | 条件保留 | 逐个核对来源、许可证和兼容性 |
| BetterGI 联动 | 条件保留 | 用户未排除；Phase 3 前审计实际调用、路径和退出行为 |

注入和插件可能触发游戏反作弊、账号处罚或稳定性风险。程序必须展示风险提示、提供纯净启动，并且不实现规避检测或隐藏注入行为。

## 6. 程序体验与维护

| 功能 | 决策 | 本项目处理 |
|---|---:|---|
| 暗色/浅色/跟随系统 | 简化保留 | 首版暗色为主，可跟随系统；不做复杂皮肤 |
| 多语言 | 保留 | 首版简中+英文，资源结构支持扩展 |
| Mica/Acrylic/视频背景 | 排除 | 仅 DWM 暗色标题栏和纯色面板 |
| 启动声音/自定义声音 | 保留 | 使用轻量 Windows 音频 API；失败不影响启动 |
| 最小化到托盘 | 保留 | 有明确退出入口 |
| 保存窗口尺寸 | 保留 | DPI/显示器变化时校正到可见区域 |
| 最小窗口尺寸 | 保留 | 防止布局不可用 |
| 应用进程优先级 | 保留但限制 | 不提供 Realtime；失败只提示不崩溃 |
| CPU 异常提醒 | 保留 | 后台低频采样，可关闭 |
| 内存定时“优化” | 不照搬 | Go 运行时下以诊断和泄漏修复代替强制 trim |
| 网络状态监控 | 按需保留 | 仅下载/更新页面需要，不常驻轮询 |
| 本程序更新 | 保留 | 签名清单、哈希、替换助手、回滚 |
| 诊断信息和日志 | 保留并强化 | 默认脱敏，不记录账号/Cookie（本项目也不采集） |
| WebView2 缓存清理 | 排除 | 项目不引入 WebView2 |
| 用户协议/首次登录页 | 排除 | 可保留简短风险说明，不做登录 |

## 7. 需要在对应 Phase 前再次确认的项目

以下不是范围含糊，而是必须以当时上游源码和外部依赖实物做技术审计：

- 每个将来安装的注入模块仍须单独确认真实开源仓库、许可证、x64 ABI、版本与 SHA-256；上游自带不透明二进制已在 S09 判定不采用。
- 每个随包插件的来源、许可证、哈希、配置格式和更新端点。
- BetterGI 联动的实际调用方式、外部程序生命周期和许可证；默认仍在范围内，但审计不通过时必须回报再决定。
- 官方下载/预下载 API 和 manifest schema 是否发生变化。
- 官服/B服切换涉及的 SDK 文件清单和可逆性。
