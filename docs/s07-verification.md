# S07 区服、本地增强与外部联动验收记录

状态：已完成  
日期：2026-07-17

## 已实现范围

- 国服官服/B 服快速切换：保留 `config.ini` 未知节和 `game_version`，从官方渠道接口取得 SDK，校验声明大小与 MD5，使用防 Zip Slip、防链接、重复路径和解压大小上限的解包器。
- 国服/国际服高级转换：分别使用当前官方 HYP/Sophon 分支和构建接口；先生成目标清单差异并在 staging 中完整校验，再统一提交数据目录改名、EXE 改名、资源替换、配置更新和渠道 SDK 清理。
- 事务恢复：文件替换、已审计删除和文件/目录改名共用持久化反向日志；提交失败或进程异常退出后可以逆序恢复。
- HDR：严格读取 HKCU `Software\miHoYo\原神` 下 DWORD 与二进制 JSON，持久化原始字节备份后才写入，多值写入中途失败自动恢复。
- 启动声音：仅接受 1 字节到 32 MiB 的 WAV，通过 WinMM 异步播放，可试听、停用并在程序退出时停止。
- BetterGI：审计 `bettergi` URL handler 和目标 EXE 后才调用 `bettergi://start`；延迟可取消；缺失或失败只写日志，不影响游戏；绝不按进程名结束非本程序所有的实例。

## 与上游实现的安全差异

上游高级转换会先递归删除整个 `Plugins`，随后直接移动目录和逐文件覆盖，失败后依赖另一次在线修复。本项目不复制该流程：只删除明确审计的渠道 SDK 文件，所有正式目录变更都进入同一事务。

## 自动化验证

- B 服 SDK API/ZIP fixture：验证严格 schema、MD5、解压限制、配置保留和事务安装。
- 高级转换 fixture：验证国服路径到国际服路径的映射复用、差异替换、目标版本配置和未知 INI 节保留。
- 故障注入：删除后遇锁文件会恢复被删文件；目录改名后遇锁文件会恢复原目录名和内容。
- HDR fake registry：验证未知 JSON 字段保留、错误注册表类型拒写、第二值写入失败恢复和备份恢复。
- BetterGI 命令解析、HDR 备份文件以及现有 S01～S06 回归测试继续执行。
- `scripts/test.ps1 -Race`：普通测试、竞态测试、`go vet`、格式和 diff whitespace 门禁全部通过。
- `scripts/build.ps1`：成功生成 `0.6.0` Console Debug 与 Windows GUI 两种 EXE。
- `scripts/verify-artifact.ps1`：GUI/Console subsystem、`FileVersion=0.6.0.0`、`ProductVersion=0.6.0`、图标和便携 data 布局全部通过。
- Win32 截图检查：`build/s07-server.png` 与 `build/s07-local.png` 在 DPI 96 下布局完整，无裁切或控件重叠。
